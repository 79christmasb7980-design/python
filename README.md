# 블럭깨기 게임 (Python + Pygame)

간단한 브릭 브레이커(블럭깨기) 게임 샘플입니다. Pygame을 사용합니다.

필요조건
- Python 3.8+
- pygame

설치 (Windows, PowerShell)

```powershell
python -m pip install --upgrade pip
pip install -r requirements.txt
```

실행

```powershell
python "블럭꺠기 게임.py"
```

조작
- ← → 또는 A/D : 패들 이동
- 마우스로 패들 이동 가능
- SPACE : 시작 / 일시정지 / 재시작

간단한 설명
- 외부 리소스(이미지 등) 없이 직사각형/원형 도형으로 그려지는 간단한 예제입니다.

문제 발생 시
- pygame 설치 관련 에러가 발생하면, 명령어 `pip install pygame`로 설치하세요.
